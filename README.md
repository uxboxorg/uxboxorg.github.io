# uxboxorg.github.io

Portfolio for [uxbox.org](https://uxbox.org) — selected 2025 / 2026 website design work.

Served as the org-level GitHub Pages site at <https://uxboxorg.github.io/>.

## Structure

```
/                           Portfolio (index.html)
/assets/                    Logo + favicon
/thumbs/                    Project thumbnails (JPEG, 1600px max)
/projects/                  Self-contained project prototypes (HTML)
```

## Local preview

```sh
python3 -m http.server 4848
# open http://localhost:4848/
```
